# 📹 Video Learning Assistant

An AI-powered application that transforms videos into an interactive learning experience.

## ✨ Features

- Analyze YouTube videos
- Upload local video files
- Automatic transcript extraction
- AI-generated summaries
- Ask questions about video content
- Semantic search using embeddings
- Fast and context-aware responses
- hosted vector storage with cloud Chroma

## 🛠 Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS

### Backend
- Node.js
- Express.js

### AI & LLMs
- Google Gemini
- LlamaIndex

### Vector Database
- Cloud ChromaDB(hosted db)

### Media Processing
- YouTube Transcript API
- yt-dlp
- FFmpeg
